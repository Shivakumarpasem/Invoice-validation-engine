# Screenshots for the README

Put your CLI report screenshots here so the images in the main README resolve.

Expected filenames (referenced from the top-level `README.md`):

- `report-headline.png`  — the SUMMARY + "fuzzy vs naive exact-match" headline.
- `report-breakdown.png` — the BREAKDOWN BY PROBLEM TYPE + TOP CASES sections.

To capture them:

1. Run the report:  `python -m invoice_engine.reporting.run`
2. Screenshot the relevant part of the terminal.
3. Save the images here with the names above (or rename and update the paths in
   the main README).
